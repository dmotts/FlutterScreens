## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/The-Young-Programer/FlutterScreens.git
   ```

2. Navigate to the project directory:
   ```bash
   cd FlutterScreens
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
